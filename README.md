ClassChat Assignment
-------------------

Requirements:
Python 3 installed

TASK01:
Run server first:
python server.py

Then run client:
python client.py


TASK02:
Run server:
python server.py

Open multiple terminals and run:
python client.py


TASK03:
Run server:
python server.py

Run clients:
python client.py


WEB VERSION:
Run WebSocket server:
python ws_server.py

Then open login.html in browser
or run:
python -m http.server 5500
and open:
http://localhost:5500/login.html


Note:
For demonstration on different computers,
change HOST to the server machine's IP address.
